# Curriculum Map (Gamified Path)

## Level 1: Communication Foundations
### Module: Google Workspace Email Setup
- Objective: Configure business Gmail correctly.
- Required actions: Create mailbox, set signature, confirm SPF/DKIM checklist.
- Validation: 80%+ quiz + checklist proof.
- Unlock: Pass module.

### Module: Microsoft 365 Email Setup
- Objective: Configure Outlook mailbox and business profile.
- Required actions: Account setup, MFA, signature, calendar and alias.
- Validation: Scenario quiz + setup checklist.
- Unlock: Pass module.

## Level 2: Web Presence
### Module: WordPress Website Starter
- Objective: Publish a basic business website.
- Required actions: Configure homepage, services page, contact form.
- Validation: URL review + configuration checklist.
- Unlock: Pass module.

### Module: Google Business Profile
- Objective: Publish and optimize local business listing.
- Required actions: Add hours, services, categories, imagery.
- Validation: Checklist + short assessment.
- Unlock: Pass module.

## Level 3: Optimization & Automation
### Module: Plugin Installation & Security Baseline
- Objective: Install core operational plugins responsibly.
- Required actions: Install security, backup, performance plugins.
- Validation: Safety quiz + plugin-state checklist.

### Module: SEO Fundamentals
- Objective: Implement on-page SEO correctly.
- Required actions: Metadata, heading structure, internal links.
- Validation: SEO rubric + quiz.

### Module: CRM Onboarding
- Objective: Connect lead capture with CRM workflows.
- Required actions: Build pipeline, import contacts, create automation.
- Validation: Process walkthrough + scenario quiz.

## Level 4: Growth Engine
### Module: Digital Marketing Launch
- Objective: Launch first cross-channel campaign.
- Required actions: define audience, craft creatives, launch plan.
- Validation: Campaign brief rubric + performance KPI quiz.

## XP, Badges, and Completion
- Each completed module grants 100 XP.
- First-pass completion bonus: 25 XP.
- Badge awarded for every level completion.
- Graduation threshold: 850 XP and all modules passed.
