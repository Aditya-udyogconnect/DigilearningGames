# MVP Definition

## Mandatory capabilities
- Playable learner dashboard with level and XP updates.
- Mission validation form with pass/fail decisioning.
- Module list with lock/unlock state visualization.
- Plugin REST endpoints for module fetch and submission validation.
- Persistent user progress (XP + completed module count).

## Done criteria
- Theme activates with no PHP syntax errors.
- Plugin activates with no PHP syntax errors.
- Dashboard and mission templates render and run JS interactions.
- Validation endpoint returns structured JSON response.
